게임 방법

  1. 난이도 선택 → 시작·목표 페이지 자동 배정
    또는 직접 원하는 페이지 입력
  2. 내부 링크만 클릭해서 목표 페이지까지 이동
  3. 최단 시간 · 최소 클릭으로 도달하면 승리!
  4. 뒤로 가기 · 검색 · 외부 링크 사용 금지
